# Dashboard

This folder contains the interactive dashboard created to visualize key insights derived from the analysis.

## Dashboard Details
- Tool used: Tableau
- Visualizations include sales trends, category-wise performance, and regional analysis

## Purpose
The dashboard provides a high-level summary of insights to support data-driven decision-making and stakeholder communication.
